Desafio de angular Cooperativa Central Ailos

Esta é a página inicial do desafio de Angular proposto pela Cooperativa Central Ailos. O projeto foi desenvolvido utilizando Angular 17 e tem como objetivo demonstrar habilidades em desenvolvimento front-end. O layout é responsivo e foi projetado para ser intuitivo e fácil de usar. O design é limpo e moderno, com uma paleta de cores agradável.
